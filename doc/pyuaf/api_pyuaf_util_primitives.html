pyuaf.util.primitives¶

This module defines some primitive data types (since Python does not differentiate clearly between, for example, an UInt32 and an Int16 value.

class BooleanArray¶

An array of Boolean instances.

You can also specify an array of Booleans with a python list of Boolean instances. But, in case you want an empty list, then you need to use BooleanArray because an empty python list does not contain any type information of the contents of the list.
