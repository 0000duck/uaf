pyuaf.client.configs¶

In UAF terminology, a Config contains one or more Settings (which are defined in the pyuaf.client.settings module).

class CreateMonitoredDataConfig¶

class pyuaf.client.configs.CreateMonitoredDataConfig¶

A CreateMonitoredDataConfig contains all service-related settings to process a CreateMonitoredDataRequest.

It contains:

  • a CreateMonitoredDataSettings instance, containing the settings for the actual CreateMonitoredData service.
  • a TranslateBrowsePathsToNodeIdsSettings instance, that will only be used to resolve the addresses that are part of the CreateMonitoredDataRequest (if needed).

  • Methods:

    __init__()¶

    Create a new CreateMonitoredDataConfig object.

    __str__(indent='', colon=40)¶

    Get a formatted string representation of the config.

  • Attributes

    serviceSettings¶

    The settings that will be used for the CreateMonitoredData service invocation.

    This attribute is a CreateMonitoredDataSettings instance.

    translationSettings¶

    The settings that will be used if the CreateMonitoredData request contains addresses that have relative paths that need to be translated.

    This attribute is a TranslateBrowsePathsToNodeIdsSettings instance.

class SessionConfig¶

class pyuaf.client.configs.SessionConfig(*args)¶

A SessionConfig instance can be used to configure one or more sessions that need to be created, in order to successfully invoke an OPC UA service.

See also

Check out example How to change the default session behavior? for more information.

  • Methods:

    __init__(*args)¶

    Create a new SessionConfig object.

    Parameters:defaultSettings (SessionSettings) – Optional parameter: the default session settings.
    __str__(indent='', colon=40)¶

    Get a formatted string representation of the config.

  • Attributes

    defaultSessionSettings¶

    The session settings that should be used for any session that needs to be created, and that is not specified (via its server URI) by the specificSessionSettings.

    This attribute is a SessionSettings instance.

    specificSessionSettings¶

    A container that maps <server URIs> (of type str) to <session settings> (of type SessionSettings).

    When sessions need to be created to a server, the UAF will first check if the URI of this server is specified by this container. If so, then the corresponding “specific” session settings will be used. If not, then the defaultSessionSettings will be used.

    This attribute is a SpecificSessionSettings instance.

    Check out the example mentioned above for more info.

class TranslateBrowsePathsToNodeIdsConfig¶

class pyuaf.client.configs.TranslateBrowsePathsToNodeIdsConfig¶

A TranslateBrowsePathsToNodeIdsConfig contains all service-related settings to process a TranslateBrowsePathsToNodeIdsRequest.

It contains two TranslateBrowsePathsToNodeIdsSettings instances, but only the translationSettings will be used.

  • Methods:

    __init__()¶

    Create a new TranslateBrowsePathsToNodeIdsConfig object.

    __str__(indent='', colon=40)¶

    Get a formatted string representation of the config.

  • Attributes

    serviceSettings¶

    Settings that won’t be used, since the UAF will use the translationSettings in the first stage (the address resolution stage) of the processing of the request. This attribute is only here since the TranslateBrowsePathsToNodeIdsConfig is implementing the same C++ template class that is used by the other service configs.

    This attribute is a TranslateBrowsePathsToNodeIdsSettings instance.

    translationSettings¶

    The settings that will be used if the TranslateBrowsePathsToNodeIds request contains addresses that have relative paths that need to be translated.

    This attribute is a TranslateBrowsePathsToNodeIdsSettings instance.
